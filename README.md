# Unique-Adventures
A project for my CIST2746 Class related to content generation algorithms and agents

To create your own build of the project, you just need to do the following

1. Open the project in the unity editor

2. Select File, then go to Build Settings

3. If there are no scenes in the "Scenes to build" box, click on "Add open scenes"

4. Under the "Platform" Section, Click on the Pc, Mac, & Linux Standalone

5. Click the "Build" button

6. Select a build destination (make a new folder to contain the build) and enjoy
